A Select Element and Much More....

SuperSelector is A JQuery UI enhancement of the standard HTML Select element.

Simply include the library in your project and you're away!

As well as giving you complete CSS control over your select elements, SuperSelector comes with a host of features to make manimupation your select elements as easy as possible.

Features

CSS Styling

Full CSS of all elements

Grouping

Group list elements togeather

Headings

List group headings

Disable

Disable list elements
