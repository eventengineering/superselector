Setup

Setting up superselector could not be simpler.

Include the library in your project and you are good to go!

include("SuperSelector.php");

Convert A Unit

To convert a unit, create an instance of SuperSelector passsing two arguments to the constructor. The first argument is the value to be converted from, the second argument is a case sensitive string representing its unit. This returns a SuperSelector object that will refrenece all its conversions to your initial unit and value.

You can then call the to() function on the SuperSelector object, passing the desired output unit as the first argument.

$demoSuperSelector = new SuperSelector(10, "m");
$demoSuperSelector->to("ft"); //returns converted value

Result: 10 Meters = 32.808398950131 Feet

A full list of available units can be found in the Available Units section.
